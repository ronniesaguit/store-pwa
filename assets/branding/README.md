Drop the branding files for the HubSuite family in this folder using these names:

- `hubsuite.svg`
- `hubsuite-trial.svg`
- `negosyo-hub.svg`
- `business-hub.svg`
- `nexora-hub.svg`

The frontend already references these paths through `store-pwa/hubsuite.js`.
If a file is missing, the UI falls back to text automatically.
